# EdTech_PM_game
Demonstration of MVP of educational game for project managers. Inspired by card games

Demo of gameplay: https://youtu.be/aaxc3vgmWiY

Опис:
Це карткова стратегія з елементами управління проєктами.
Головна ціль - досягнення необхідної кількості балів для побудови «розумного міста».
Загальні правила:
- Для перемоги потрібно набрати 80 балів, що розділені на 4 рівні міста.
На реалізацію проєкту розумного міста надається 100 ігрових днів.
Початковий бюджет 700 грошових одиниць.
- Для побудови міста потрібно виконувати проєкти. На проєкти призначаються ресурси, які гравець наймає з ринку праці. 
Під час гри гравець може покращувати свій офіс та призначати манеджерів, що є однією з умов для успішного завершення гри.
- У грі присутня спеціалізація у проєктів та ресурсів, а саме: наука, логістика, медицина, енергетика, економіка та державні.

Гейміфікація представлена двома складовими. 
- Перша - менеджмент ресурсів, що представлений необхідністю обирати оптимальні проєкти для виконання, вчасно назначати менеджерів та вміти адаптуватися до непередбачуваних умов. При необдуманому розподілу наявних ресурсів у гравця є ймовірність стати банкрутом або витратити всі 100 днів і не встигнути збудувати місто.
- Друга  – це тести з запитаннями про управління проєктами. Через певний, наперед визначений грою, проміжок часу гравцю задається питання, пов’язане з управлінням проєктами. Залежно від відповіді гравець отримає позитивний або негативний ефект для гри.
